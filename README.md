# 🌿 Farm Fresh Vegetables Marketplace

A modern, responsive farm-to-consumer platform where users can buy fresh vegetables directly from local farmers.

This project focuses on simplicity, trust, and freshness by providing a clean interface for discovering, purchasing, and subscribing to daily vegetable needs.

---

## 🚀 Features

* 🥬 Browse fresh vegetables from nearby farmers
* 🔍 Smart search and filtering
* 🛒 Add to cart with real-time updates
* 🔁 Daily subscription (Dailies system)
* 📦 Bulk and pre-order options
* 💳 Secure checkout (UPI / Card / COD)
* 📍 Order tracking with live status
* 📜 Order history with quick reorder

---

## 🎯 Key Highlights

* Direct farmer-to-customer connection
* Focused on vegetables only (no clutter)
* Clean and minimal UI/UX
* Fully responsive (mobile, tablet, desktop)
* Smooth user experience with modern design principles

The idea behind this project is to simplify how users buy vegetables while supporting local farmers.
